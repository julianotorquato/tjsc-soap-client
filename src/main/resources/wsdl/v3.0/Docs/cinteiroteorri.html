Type: nsS:AtoSelado
Containing Schema: xsCInteiroTeorRI.xsd
Abstract
Documentation: 
Ato destinado a contemplar as informações por ocasião da determinação judicial 
de cancelamento de matrícula, ou outro tipo de cancelamento que não se trate do encerramento 
legal/regimental de uma matrícula (para o qual existe o ato Certidão de Encerramento de Matrícula).
