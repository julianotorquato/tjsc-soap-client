Name: filiacao

Type: nsS:PessoaFiliacao
Containing Schema: xsCNascimento.xsd
MinOccurs 1
MaxOccurs unbounded
Abstract
Documentation: 
Neste campo deve-se informar a filiação do nascido. O campo 
foi especializado para representar o modelo pai(mãe)/avós, porém livre da 
definição de gênero. Reflete, portanto, uma filiação e seus progenitores. 
Para cada filiação é necessário relacionar ao menos um progenitor – é possível 
relacionar tantos quantos existirem. Deve ser preenchido pelo menos um.
